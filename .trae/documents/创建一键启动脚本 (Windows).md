### 脚本功能计划
1.  **前端自动化**：
    *   检查 `web/node_modules` 是否存在。
    *   如果不存在，自动执行 `npm install` 安装依赖。
    *   在新窗口中启动前端开发服务器 (`npm run dev`)。
2.  **后端自动化**：
    *   检查 `backend/.env` 配置文件是否存在，若不存在则自动从模板复制。
    *   检查 `backend/.venv` 虚拟环境是否存在，若不存在则自动创建。
    *   自动激活虚拟环境并安装/更新依赖 (`pip install -r requirements.txt`)。
    *   在新窗口中启动后端 API 服务 (`uvicorn main:app`)。

### 预期效果
您只需要双击这个 `start_dev.bat` 文件：
*   它会自动帮您准备好所有开发环境。
*   它会弹出两个新窗口，分别运行前端和后端。
*   您无需记忆任何命令行指令。
