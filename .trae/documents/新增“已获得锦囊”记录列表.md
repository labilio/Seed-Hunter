我将为您在“锦囊”页面下方新增“已获得锦囊”列表。

### 1. 新增数据状态
目前代码中购买锦囊后直接将内容显示在对话框，没有记录购买历史。为了实现列表展示，我需要：
*   新增 `purchasedHints` 状态：`const [purchasedHints, setPurchasedHints] = useState([])`
*   数据结构示例：
    ```javascript
    [
      { id: 1, title: '初级锦囊', level: 2, content: '...', timestamp: 1234567890 }
    ]
    ```

### 2. 购买逻辑修改
*   修改 `handlePurchaseHint` 函数：
    *   在扣除积分的同时，将当前购买的锦囊信息（标题、当前关卡、内容）添加到 `purchasedHints` 数组中。

### 3. UI 实现（已获得锦囊）
*   **位置**：在 `aside` 的锦囊 Tab 页中，紧接在现有锦囊商品列表下方。
*   **分隔线**：仅当 `purchasedHints.length > 0` 时，显示一条 `h-px bg-black/10` 的灰线。
*   **列表样式**：
    *   紧凑布局，控制高度。
    *   每行显示：左侧“锦囊名称” + “Lv.X”标签；右侧小巧的“查看”按钮。
    *   点击“查看”：弹窗显示该锦囊的详细内容（复用现有的 Modal 或简单的 alert，建议复用 Modal 以保持美观）。

### 4. 弹窗逻辑扩展
*   扩展 `hintModal` 或新增一个查看状态，用于展示已购买锦囊的具体内容。

这个方案既满足了“记录”需求，又符合“不占用过多空间”的视觉要求。
